# Codestyle: HTML

## Обязательные элементы
```html
<!-- Тип документа HTML5 -->
<!DOCTYPE html>
<!-- Корневой "html" элемент с указанием языка -->
<html lang="en">
  <!-- Метаинформация о документе -->
  <head>
    <!-- Кодировка документа -->
    <meta charset="utf-8">
    <!-- Настройка области видимости -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Заголовок -->
    <title>...</title>
  </head>
  <!-- Тело документа -->
  <body>...</body>
</html>
```

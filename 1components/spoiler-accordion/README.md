# Спойлер

## HTML-разметка
**Тег**
- Используется тег `<details>`


## CSS-стилизация
**Скрыть стандартный маркер**
- Для элемента `<summary>` скрыть стандартный маркер (стрелочку) через CSS
- Пример: `list-style: none`
